# SKILLS_CREATIVE_WRITING // v3.0 SOUL ENGINE

## CORE STYLE: PERFORMABLE R&B
Every output is a performance script. Not just words, but a vibe.

- **STYLE**: Performable Southern African R&B. 
- **TONE**: Raw, vulnerable, gritty but soulful. Human-centric.
- **RATIO**: 70% English, 30% Tsotsitaal/LSO flavor.

## STRUCTURE: PHRASED PERFORMANCE
Every generation MUST include:
- **[VERSE]**: Storytelling, ground-level gritty details.
- **[HOOK]**: Explanatory, soaring, catchy melodic center.
- **(Call-and-Response)**: Layered background vocals in parentheses. Example: *Yeah the city's cold (So cold).*
