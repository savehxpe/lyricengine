# FEATURES // MODULAR CAPABILITIES

## 1. [MAIN] Library (Writing Studio)
- **Primary Input:** Real-time text area for raw ideas.
- **Primary Output:** The processed engine result with active highlighting.
- **Action Buttons:**
    - [TRANSFORM]: Full rewrite based on IDENTITY.md.
    - [REMIX]: Change the meaning or narrative angle.
    - [FREESTYLE]: Infinite generation from a single seed word.

## 2. History (The Archive)
- **Saved Blocks:** Auto-saves every [TRANSFORM] or [REMIX] output.
- **Tracking:** Timestamped entries (e.g., `2026-03-29 // 01:13`).
- **Version Control:** Toggle between V1, V2, and User Edits.
- **Quick Load:** Click any history item to push it back into the Library.

## 3. Settings (Control Panel)
- **Rhyme Intensity (0-100%):** Controls the density of internal rhyme chains.
- **Outsider Intensity (0-100%):** Controls the level of abstract/raw/metaphorical language.
- **Localization (0-100%):** Controls Lesotho cultural references and linguistic flavor.
- **Vibe Toggles:** Dark Mode (default), High Contrast, Mono Mono.

## 4. RMD Database (Rhyme Memory)
- **Pattern Storage:** Stores successful multisyllabic schemes (e.g., `O-A-O-A` endings).
- **Structures:** Reusable bar structures (e.g., "The 16-Bar Brutalist").
- **Writing Memory:** Learns user-preferred vocabulary and adds it to the weighting logic.
